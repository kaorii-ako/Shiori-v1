# Shiori v1

An AI-assisted productivity desktop app that connects to Google Classroom, Gmail, and Calendar to automatically organize assignments, extract deadlines, and generate study plans.

## ✨ Features

- 🤖 **AI-Powered Organization** - Automatically organize and prioritize your tasks
- 📚 **Google Classroom Integration** - Sync assignments and deadlines
- 📧 **Gmail Integration** - Extract important information and deadlines from emails
- 📅 **Google Calendar Integration** - Seamlessly manage your schedule
- 📋 **Smart Assignment Organization** - Automatically categorize and organize your work
- ⏰ **Deadline Extraction** - Never miss an important deadline
- 📊 **AI-Generated Study Plans** - Get personalized study recommendations

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- Google API credentials (for Classroom, Gmail, and Calendar)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/kaorii-ako/Shiori-v1.git
cd Shiori-v1
```

2. Install dependencies:
```bash
npm install
```

3. Set up Google API credentials:
   - Visit [Google Cloud Console](https://console.cloud.google.com/)
   - Create a new project
   - Enable Classroom, Gmail, and Calendar APIs
   - Create OAuth 2.0 credentials
   - Add your credentials to the application

4. Start the application:
```bash
npm start
```

## 🎯 How It Works

1. Connect your Google accounts (Classroom, Gmail, Calendar)
2. The app automatically syncs and analyzes your assignments and deadlines
3. AI processes the information to extract key dates and requirements
4. Personalized study plans are generated based on your workload
5. Receive smart recommendations for task prioritization
